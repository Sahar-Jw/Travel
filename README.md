# 🌍 Travel - Best Travel Agent Website

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

## 📖 About This Website
**Travel** is a modern, responsive landing page for a premium travel agency. It promotes global travel experiences, helping users discover destinations, learn why to choose this agent, and get in touch for memorable trips. Built as a static single-page site, it's perfect for quick deployment and showcases services like guided tours, special tickets, and 24/7 support.

> \"Travel around the world to see the beauty of nature easily and safely with Travel.\"

## ✨ Features
- **Hero Section**: Eye-catching banner with world travel tagline and promotional text.
- **Destination Search**: Interactive search bar and grid of featured destinations (e.g., Mirror Lake, Indonesia – customizable placeholders).
- **Why Choose Us**: 3 highlight cards on best guides, special tickets, and 24/7 support.
- **Memorable Trips**: Promotional banner with CTAs (\"See More\", \"Discover More\").
- **Trip Documentation**: Video placeholder for trip highlights.
- **Best Offers**: Promo section with call-to-action button.
- **Responsive Design**: Mobile-friendly (collapses to single-column on <=992px).
- **Interactive Elements**: Bootstrap offcanvas mobile menu, hover effects/transitions, scroll-to-top.
- **Footer**: Social links (Instagram, Twitter, YouTube), quick nav.

## 🛠️ Technologies & Programs Used
| Category | Technologies |
|----------|--------------|
| **Structure** | HTML5 (semantic elements) |
| **Styling** | Bootstrap 5.3 (responsive grid/navbar), Custom CSS (SCSS-like: CSS variables, nesting, media queries) |
| **Fonts** | Google Fonts: [Gilda Display](https://fonts.google.com/specimen/Gilda+Display) (headings), [Urbanist](https://fonts.google.com/specimen/Urbanist) (body) |
| **Assets** | SVG icons, PNG images for hero/destinations |
| **JavaScript** | Bootstrap JS bundle (for offcanvas/interactivity) |
| **Deployment** | Static site – no build tools needed |

All dependencies loaded via CDN – zero setup!
